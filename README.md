# 🏦 Loan Approval Prediction System

## Overview
A Machine Learning project that predicts whether a loan application will be approved based on income, credit score, loan amount, and age.

## Features
- Predict loan approval status
- User-friendly Streamlit UI
- Real-world banking use case

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Project Structure

loan-approval-prediction/
│
├── app.py
├── train_model.py
├── loan.csv
├── model.pkl
├── requirements.txt
└── README.md

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
